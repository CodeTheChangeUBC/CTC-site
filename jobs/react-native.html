React Native Developer | Deadline: September 18th, 11:59pm

Are you looking to build on your current coding skills? Do you want to give back to the local community, and make new friends while doing so? Then we want you to join the Code the Change Foundation!

The Foundation (formerly Code the Change UBC) builds open-source software solutions to enhance the effectiveness of non-profit organizations who don't have the means to develop it themselves. We believe in the powerful unification of students' passions and technical skills with the vision and knowledge of philanthropic organizations.

Currently, we are partnering with the University Health Network to build their cross platform Mobile Application. We are looking for 5 curious, passionate, and driven people to join our Foundation as React Native Developers!

To learn more about the organization check out the University Health Network.

Description & Responsibilites:
  • Collaborate in a team of ~5-6 people to develop a cross platform mobile application for the University Health Network
  • Work with React Native modules
Availability:
  • Job term date: September 28, 2019 to April 30, 2020
  • Be able to attend mandatory weekly team meetings on Saturdays from 12:00pm-3:00pm at UBC Vancouver Campus
  • Be able to commit to your project 3 hours a week, outside of weekly team meetings
What you need to have:
  • Be a team player, as you may be working with the same team for 8 months
  • Be open to new ideas and constructive criticism, while working in a collaborative environment
  • Be able to estimate task timelines and manage time effectively
  • Passionate about giving back to the local community
  • Bonus if you have worked on open-source software solutions in the past
Basic technical requirements:
  • Experience with and/or willingness to learn React Native and JavaScript
  • Experience building mobile applications
  • Experience integrating web services and RESTful APIs
  • Proficient understanding of code versioning tools, such as Git
How to apply:

Once you have read over this job description, please fill out the form below in order to apply. Be ready to submit your resume.

The application deadline is: September 18th, 11:59pm

Interviews will be conducted on a rolling basis.

If you have any questions, please email us.
